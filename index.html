TailwindCSS

TailwindCSS เป็น utility-first CSS framework ที่ช่วยให้คุณสร้างการออกแบบที่กำหนดเองได้อย่างรวดเร็วโดยไม่ต้องออกจาก HTML ของคุณ ช่วยให้การพัฒนาเว็บไซต์เป็นไปอย่างรวดเร็วและมีประสิทธิภาพ

GitHub Pages

GitHub Pages เป็นบริการโฮสติ้งสำหรับเว็บไซต์สถิตที่ดึงไฟล์ HTML, CSS และ JavaScript โดยตรงจาก GitHub repository ของคุณ ทำให้การเผยแพร่เว็บไซต์เป็นเรื่องง่าย

การเริ่มต้น

เริ่มต้นด้วยการสร้าง repository ใหม่บน GitHub จากนั้นอัปโหลดไฟล์ของคุณและเปิดใช้งาน GitHub Pages ในการตั้งค่า repository เพียงเท่านี้คุณก็มีเว็บไซต์ออนไลน์แล้ว
